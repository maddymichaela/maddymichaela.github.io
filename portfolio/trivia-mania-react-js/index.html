Project Overview

In this project, I worked with my teammates, Catherine, James, Jay and Anu to create a mobile-responsive React.js app. 

In this project, I was mainly in charge of designing the app’s logo, prototype with Figma, and coding the overall design with HTML, CSS, and React/JSX. 

Keeping in mind the user journey that the user will take to navigate the web app, we’ve implemented a sleek and intuitive user interface to enhance the overall user experience, ensuring seamless navigation and engagement while climbing to the top of the leadership board.
